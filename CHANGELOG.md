# 更新日志

所有重要的项目更改都将记录在此文件中。

## [未发布]

### 新增
- 添加帖子详情页面用户名正确显示功能

- 修复时间显示为相对时间的问题，改为详细时间格式
- 修复第二页及后续页面用户名显示错位的问题

- 优化用户名提取逻辑，支持多种查找方式
- 改进错误处理，保留关键错误日志

## 版本号规则

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

## 更新类型标记

- **新增** - 新功能
- **修复** - 问题修复
- **优化** - 性能优化或代码改进
- **删除** - 功能删除
- **安全** - 安全相关更新
