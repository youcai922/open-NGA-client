# 更新日志

所有重要的项目更改都将记录在此文件中。

## [0.1.2] - 2026-03-11

### 新增
- 添加帖子详情分页功能
  - 分页器固定在页面底部
  - 支持上一页/下一页/点击页码跳转
  - 支持快速跳转输入框
  - 显示当前页/总页数（如 1/5）
- 添加准确的楼层计算，页面跳转时楼层正确显示
  - 第1页：从1楼开始
  - 第2页：从21楼开始
  - 以此类推...

### 修复
- 修复第二页及后续页面回复时间显示错误的问题
  - 之前：显示当前时间
  - 现在：正确显示实际回复时间
  - 优化时间元素提取逻辑，从对应楼层的位置查找时间信息
- 修复总页数计算不准确的问题
  - 优先从 NGA 的 `__PAGE` JavaScript 变量中提取总页数
  - 添加备用提取方式（分页链接、末页文本）

### 优化
- 保留滚动懒加载功能，与分页功能共存
- 分页器靠右显示，更符合常规布局
- 清理所有调试日志输出

## [0.1.1] - 2026-03-10

### 新增
- 添加首页项目信息弹窗功能
- 添加帖子详情页面用户名正确显示功能
- 添加 HTML 实体编码解码支持（emoji 和特殊字符）
- 添加详细的日期时间显示格式（YYYY-MM-DD HH:mm）

### 修复
- 修复用户名正确显示功能
- 修复 HTML 实体编码解码问题（emoji 和特殊字符）
- 修复帖子详情页面用户名显示为 ID 的问题
  - 从用户链接的 textContent 中正确提取用户名
  - 从 JavaScript 变量中提取用户名映射
  - 添加智能查找逻辑处理 postauthor ID 不连续的问题
- 修复时间显示为相对时间的问题，改为详细时间格式
- 修复第二页及后续页面用户名显示错位的问题

### 优化
- 清理所有调试日志输出
- 优化用户名提取逻辑，支持多种查找方式
- 改进错误处理，保留关键错误日志

## [0.1.0] - 2026-03-04

### 新增
- 初始版本发布
- 支持查看 NGA 论坛板块列表
- 支持查看主题列表
- 支持查看帖子详情和回复
- 支持滚动加载更多内容
- 支持 NGA 表情转换显示
- 支持图片代理加载
- 支持引用块解析
- 支持收藏板块功能

## 版本号规则

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

## 更新类型标记

- **新增** - 新功能
- **修复** - 问题修复
- **优化** - 性能优化或代码改进
- **删除** - 功能删除
- **安全** - 安全相关更新
